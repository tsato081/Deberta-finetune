# 新設カテゴリ

- 業績改善
    定義：
    "決算・業績予想における改善（増収増益、上方修正、黒字転換、利益率改善など）が主題の発表・報道を対象とします。
    四半期決算や通期予想の上方修正、コスト削減・構造改革による収益改善、過去最高益など、数値の裏付けがある記事を主に想定します。
    資金調達や補助金採択など「資金流入」自体が主題の場合は各カテゴリ（「資金調達」「補助金獲得」）へ、単発の受賞・認定は「受賞」「認定取得」へ回します。"

- 業績悪化
    定義：
    "決算・業績予想における悪化（減収減益、下方修正、赤字拡大、見通し不透明化など）が主題の発表・報道を対象とします。
    対象指標は売上高・営業利益・経常利益・純利益などで、四半期決算、通期予想、セグメント別の悪化、特損計上など、数値の変化が明示される記事を想定します。
    債務超過や資金繰り危機など財務状態が中心なら「債務超過」へ、倒産手続の開始が主題なら倒産カテゴリへ、単なる株価動向・市場環境の一般論は取得対象外（カテゴリ付与しない）とします。"

# 定義変更するカテゴリ
- 生産再開
    "停止していた生産活動（工場・ライン）の再開／操業再開／稼働再開が主題の発表・報道を対象とします。
    再開時期、対象拠点、復旧内容、再発防止策、供給回復の見通しなど、再開の事実と影響が具体的に書かれている記事を想定します。
    システム復旧（オンラインサービス復旧）は「システム障害」へ、サイバー攻撃からの復旧は「サイバー攻撃」へ、再開ではなく増産・能力増強が主題なら「生産拡大」へ回します。"
- イベント
    "自社主催・共催のイベント／セミナー／ウェビナー／講演会の開催告知・実施報告、または展示会・見本市への出展（参加）を主題とする発表・報道を対象とします。
    開催日時・場所、登壇者、テーマ、参加募集、出展ブース案内、イベントレポートなど、イベントとしての具体情報がある記事が典型です。
    広告・キャンペーン等の販促が主題なら「広報活動」へ、採用目的の説明会が中心なら「新規採用」へ、単なる社外表彰は「受賞」へ回します。"
- 職場環境改善
    "働き方・福利厚生・安全衛生・オフィス環境など、従業員の働きやすさを高める制度・施策の導入／拡充が主題の発表・報道を対象とします。
    リモートワーク、フレックス、休暇制度、育児介護支援、健康施策、オフィス改装など、恒常的な制度変更・施策実施を含みます。
    賃金水準の引き上げは「賃上げ」へ、DE&I方針は「ダイバーシティ」へ、単なる本社移転は「本社住所移転」へ回します。"
- 新設備導入
    "既存の工場・施設・生産ライン等への新設備・機械・装置の導入／更新／入替が主題の発表・報道を対象とします。
    導入する設備の種類（ライン、ロボット、検査装置等）、導入目的（省人化、品質向上、増産等）、導入時期、導入規模が具体的に書かれる記事が典型です。
    投資額中心で計画段階の話題は「設備投資」へ、工場そのものの新設は「新工場竣工」へ、増産・能力増強が主題なら「生産拡大」へ回します。ITツールの導入等はDXに回します。"
- 批判的投稿
    "SNS・口コミ・掲示板等での企業／商品／サービスへの批判・炎上・ネガティブ拡散が主題で、事実関係が未確定または“反応・評判”中心の記事を対象とします。
    「炎上」「批判殺到」「低評価」「拡散」など、世論や反応の説明が中心で、企業の公式発表がなく真偽が確定していないケースを含みます。
    事故・漏えい・不正など事実が確認され、企業対応が報じられている場合は各事象カテゴリ（「事故」「情報漏洩」「品質管理不正」等）を優先し、問題発言が原因なら「不適切発言」、広告表現の問題なら「不適切広告」へ回します。"
- システム障害
    "自社が提供・利用するシステム／オンラインサービスでの障害（停止・遅延・ログイン不可・不具合）が主題の発表・報道を対象とします。
    障害発生日、影響範囲、復旧見込み、原因（更新不具合、負荷、設定ミス等）、補償対応など、運用障害としての具体情報がある記事が典型です。
    攻撃起因は「サイバー攻撃」へ、漏えいが確定していれば「情報漏洩」へ、通信網の障害が主因なら「通信障害」へ回します。"
- 新商品の展開
    "新商品／新サービスの発表・発売・提供開始、または既存商品の大幅リニューアル（顧客提供価値の変更）が主題の発表・報道を対象とします。
    発売日・提供開始日、対象顧客、価格、機能・特徴、提供エリア、提携先など、顧客向けの提供内容が具体的に記載される記事を想定します。
    社内向けのIT導入や業務改革が中心なら「DX」へ、研究段階・実証段階の技術発表は「研究開発」へ、店舗の開店・施設開設が主題なら各カテゴリへ回します。"

    （ただし、おそらく決算情報公開の記事は業績好調/悪化のいずれかに振り分けられると思う）
